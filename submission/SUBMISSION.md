## 2. Description

```
ParkSense turns raw illegal-parking violation data into a ranked, explainable enforcement plan.

THE PROBLEM
On-street illegal and spillover parking near commercial areas, metro stations and event venues
chokes carriageways and intersections. Today, enforcement is reactive and patrol-based, there is
no heatmap linking violations to congestion impact, and teams cannot objectively prioritize which
zones to enforce first.

OUR SOLUTION
ParkSense is a live command dashboard that detects illegal-parking hotspots and quantifies their
impact on traffic flow, so authorities can enforce the right zones at the right time.

- Live hotspot heatmap of illegal-parking clusters across Bengaluru, colored by enforcement tier
  (Red = dispatch now, Amber = pre-emptive sweep, Green = monitor).
- Congestion Impact Score (CIS): a transparent 0–100 score that quantifies how much each hotspot
  is hurting traffic flow, built from violation pressure, time-of-day surge, road criticality,
  spillback risk and historical non-compliance.
- Enforcement Priority Queue: every zone ranked by urgency + patrol feasibility, with response
  time and a one-click Dispatch action that estimates the priority drop and commuter-minutes saved.
- 24-hour Time Simulator: scrub any hour to see how violations and congestion evolve — rush-hour
  spikes vs. clear nights — proving the model reacts to real demand.
- Automated Situation Report: a per-zone briefing generated by a deterministic rule engine
  (fully reproducible, no black box), explaining the situation and the recommended patrol.
- Engineering transparency: a "Show raw payload" toggle exposes the live JSON and the full CIS
  score breakdown, and the priority queue is exportable to CSV.

WHY IT WINS
It directly answers the theme: detect hotspots, quantify traffic-flow impact, and enable targeted
enforcement. Every number is explainable, the dashboard is real-time, and the dispatch loop closes
the gap between detection and action.

TECH STACK
FastAPI + WebSockets (Python) backend computing the CIS engine; React + Vite + Leaflet frontend
with a live heatmap. Seeded, reproducible synthetic dataset modeling real Bengaluru hotspots.
```

## 10. Instructions to Run

```
PREREQUISITES
- Python 3.10+  and  Node.js 18+

1) BACKEND (terminal 1)
   cd parksense/backend
   pip install -r requirements.txt
   python -m uvicorn main:app --host 127.0.0.1 --port 8000
   # Verify: open http://127.0.0.1:8000/health  ->  {"status":"ok"}

2) FRONTEND (terminal 2)
   cd parksense/frontend
   npm install
   npm run dev
   # Open http://localhost:5173

Notes:
- The backend auto-generates the synthetic dataset on first run.
- If the backend runs on another host/port, set VITE_API_BASE before "npm run dev"
  (Windows: set VITE_API_BASE=http://127.0.0.1:8000).
```
